# 🔒 Vault Messenger
**The Universal Privacy Layer for Web 4.0**

Vault Messenger is a high-security, client-side encryption tool designed to keep private conversations private. Built with a focus on stealth and multi-language support (English, Bengali, and more), it ensures that your data never leaves your device in a readable format.

### 🌟 Exclusive Features
* **Universal Unicode Engine:** Seamlessly encrypts Bengali, English, and Emojis.
* **Ghost Mode:** Automatically wipes the input field after encryption so no traces are left behind.
* **Privacy Shield:** The app instantly blurs if the user switches tabs or moves away, preventing shoulder-surfing.
* **Self-Destruct Timer:** Decrypted messages vanish after 30 seconds for maximum security.
* **Stealth Type:** Mask your typing in real-time, appearing as dots to anyone watching.

### 🚀 Quick Start
1. Visit the [Live Demo](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)
2. Agree on a **Secret Shift Key** (e.g., 15) with your recipient.
3. Type your message, click **ENCRYPT**, and copy the result to Messenger/WhatsApp.
4. Your recipient pastes the text, enters the same key, and clicks **DECRYPT**.
